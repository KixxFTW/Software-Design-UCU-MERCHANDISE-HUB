Security Features

  • Secure password hashing using bcrypt
    -Turns passwords into secret codes so they can't be stolen easily.

  • Session-based authentication
    -Ensures that only authorized users can access certain parts of the system.

  • CSRF protection
    -Prevents cross-site request forgery attacks, which can compromise user data.

  • Input sanitization
    -Prevents malicious input from being processed by the system.

  • Role-based access control
    - Only lets users do things they’re allowed to — like admins vs. regular users.
